== Description ==

RPW - Related posts widget displays posts, and custom post types related to the current posts, introducing your readers to other relevant content on your site.

Related posts widget is a powerful plugin for WordPress that allows you to display a list of related posts on your website.

The plugin also comes with it's very own inbuilt stylesheet that let's your related posts look awesome!

For Demo <strong><a href='http://rpw.phpfirst.in/' target='_blank' >Click Here</a></strong>

= Key features =
* **Display**: Plugin come with default 3 types of display
	* Small
	* Medium
	* Large
* **Styles**: Want more control over style? you can change widget style from an admin interface like color, fonts, number of pixels from top of the screen left and many more options, etc.

= Upcoming features =
* **Display**: In few updates tots of new options available:
	* Option to add related posts to widgetized area of your theme
	* Option to use `[RPW]` to display the related posts anywhere you want in the post content
	* Option to add an external link and/or custom article/text image
	* Option to add a subscribe form inside the widget
	* Option to add a banner inside the widget (adsense etc)
	* Option to show tweets or facebook posts instead
	* Option to add the widget on any corner or side of the screen
	* Option to find related posts by title and/or content of the current post

== Installation ==

This section describes how to install the plugin and get it working.

= Via admin: =
1. Go to Plugins -> Add New
2. Search for WordPress Related Posts
3. Install the plugin called "RPW - Related Posts Widget" and activate it
4. After activation go to setting page and set your settings.
5. Done!

= Via upload: =
1. Download the plugin .zip file
2. Log in to yourdomain.com/wp-admin
3. Click Plugins -> Add New -> Upload
4. After installation activate the plugin and check setting page.
5. You're finished!
